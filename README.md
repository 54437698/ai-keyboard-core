## 🛠 System Architecture
This project is decoupled from OS-specific "trash." The logic is stored in pure data structures to ensure low-latency performance and battery efficiency.

### 1. The UI Layer (`/ui`)
- **Number Row:** Permanent dedicated row (no long-press required).
- **Spanish Support:** Native `ñ` and accented vowel mapping (`á, é, í, ó, ú`) defined in `master_layout.json`.
- **Engine:** Designed for Jetpack Compose (Android) or SwiftUI (iOS) consumption.

### 2. The Core Engine (`/core`)
- **Logic:** C++ Bridge for direct NPU/DSP interaction.
- **Memory:** Optimized for low-footprint LLM inference (4-bit quantization).

### 3. The Forensic Log
- **Status:** Android Studio Nuked. 
- **Platform:** GitHub-First Development.
- **Protocol:** "Data Jar" sync via Google Keep.
